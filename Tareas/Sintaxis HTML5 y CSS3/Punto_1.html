Elementos basico de HTML estandar

1. Headings

Propósito: Los headings o encabezados son indicadores que guían a los lectores a través de una página. Además, los encabezados pueden ayudar a los usuarios a conocer la estructura de la página.

Sintaxis: Los headings se escriben dependiendo el tamaño para lo que se necesitan, van desde <h1>Texto</h1> hasta <h2>Texto</h2> subiendo el valor de h.

Atributos:
-style para definir tamaño, etc. -color para definir el color de la letra. -align para definir el alineamiento.

2. Paragraphs

Propósito: Se crearon para el manejo más eficiente y sencillo de párrafos, esto es un grupo de oraciones que comparten un tema.

Sintaxis: se escribe de esta forma <p >Texto </p>

Atributos:
-color para definir el color de la letra. -align para definir el alineamiento.

El párrafo está constituido por una oración principal que puede ser distinguida fácilmente, ya que enuncia la parte esencial de la cual dependen los demás. Es posible decir entonces que la oración principal posee un sentido esencial del párrafo. La oración principal puede aparecer en el texto de forma implícita o explícita. Cuando la oración principal está implícita, ésta no aparece por escrito en el párrafo y es necesario deducirla. En cambio, la explícita, sí la encontramos escrita y podemos encontrarla al principio, en el medio o al final del párrafo.

3. Links

Propósito: Se crearon para poder establecer links a páginas, cuando sea necesario, puede ser usado de esa forma o usarlo en un botón para redireccionar la página actual.

Sintaxis: <a href="https://www.facebook.com"> Esto es un link </a> intercambiando la dirección de Facebook por la dirección que se necesita.

Atributos:

Esto es un link

4. Imágenes

Propósito: Funcionan para hacer un sitio más estético, poder dar ideas sobre distintos temas y hacer un sitio mas amigable para el usuario.

Sintaxis: <img src="perro.jpg" alt="Perro"> intercambiando perro.jpg por el nombre y la ubicación de la imagen necesaria.

Atributos importantes:
-alt para dar un titulo si no se carga la imagen. -width para controlar la anchura. -height para controlar la altura.

5. Botones

Propósito: Funciona para hacer alguna acción o redireccionar a algún lugar.

Sintaxis: <button> Presióname para realizar alguna acción </button> intercambiando el texto actual por el texto que se quiere que el botón tenga escrito.

Atributos importantes:
-onclick para controlar la función del mismo -name para identificarlo -disabled para deshabilitarlo -type para escoger el tipo

6. Listas

Propósito: Creación de listas para enumerar ideas, oraciones, secciones, etc.

Sintaxis:
<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

Atributos importantes:
-href para direccionar el elemento a algún lugar -ul para hacer la lista desordenada -ol para hacerla ordenada

  1. Coffee
  2. Tea
  3. Milk

10. Divisiones o Secciones

Proposito: Es importante para la página, poder dividir la misma en secciones, por ejemplo, este sitio esta dividido en dos columnas, esto para mantener la estética.

Sintaxis: Esta funciona abriendo y cerrando la etiqueta div de esta forma <div> Todos los elementos que se quieren dividir </div>.

Atributos importantes:
-width es el ancho de la división. -padding es para controlar el espacio. -height es el alto de la división.

11. Contenedor de texto

Propósito: El contenedor de texto o span funciona para contener trozos de texto en un heading, párrafo, etc. Además, se pueden modificar estos trozos con un color, un tamaño, etc.

Sintaxis: <h1>My <span style="color:red">Important</span> Heading</h1> donde se puede ver que por ejemplo se le cambia el color al trozo de texto definido en el heading.

Atributos:
-color para controlar el color -size para controlar el tamaño

My Important Heading

12. Paginas dentro de sitios

Propósito: A veces es necesario mostrar una pagina dentro de otra página, esto puede ayudar a ahorrar tiempo y mantener o aumentar la estética del sitio.

Sintaxis: < iframe src="Imagenes/ejemplo.htm"> < /iframe> donde se puede usar un sitio alojado en la computadora o un sitio en internet, por medio del link del mismo.

Atributos importantes:
-height para escoger la altura de la ventana. -width para escoger el ancho de la ventana.

13. Modificadores de texto

Modificación del texto Propósito: Es necesario a veces usar tipos de letra distinta, subrayar texto, etc. Esto dependiendo el texto, esto para mantener la estética y crear un ambiente más amigable.

Sintaxis: todos siguen el formato de <b>This text is bold</b> donde b es el parámetro que cambia.

Atributos:
Este atributo incluye los atributos globales por lo que no posee atributos específicos. Existen distintos tipos de modificadores entre los que se puede nombrar los siguientes:
b -Bold, strong -Subrayar, i -Italica, em -Enfatizar, mark -Marcar, small – Pequeño, etc.

This text is normal.

This text is italic.

This text is emphasized.

15. Caja de opciones

Propósito: Para tener varias opciones que se pueden seleccionar sin tener que usar todo el espacio que ocuparían estas secciones.

Sintaxis:
<select>
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
</select>

Atributos:
-form para definir la forma -size para definir tamaño -multiple para poder seleccionar múltiples opciones

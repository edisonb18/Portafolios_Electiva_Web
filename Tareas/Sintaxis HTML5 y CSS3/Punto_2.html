Elementos agregados en HTML5

1. Artículo

Propósito: Define un artículo en un documento
Sintaxis:
<article>
<p>Google Chrome is a free, open-source web browser developed by Google, released in 2008.</p>
</article>

Atributos: El articulo soporta los atributos globales de HTML tales como class, dir, data, etc.

Google Chrome

Google Chrome is a free, open-source web browser developed by Google, released in 2008.

2. Aside (al lado)

Propósito: Define el contenido aparte del contenido de la página

Sintaxis:
<aside>
<h4>Epcot Center</h4>
<p>The Epcot Center is a theme park in Disney World, Florida.</p>
</aside>

Atributos: El Aside o al lado soporta los atributos globales de HTML tales como class, dir, data, etc.

My family and I visited The Epcot center this summer.

Epcot Center

The Epcot Center is a theme park in Disney World, Florida.

3. Dialog

Propósito: Define una caja de dialogo o una ventana

Sintaxis:
<dialog open>This is an open dialog window</dialog>

Atributos:
-Posee el atributo open que especifica que el elemento de diálogo está activo y que el usuario puede interactuar con él. -Posee los atributos globales de html.

En el ejemplo se coloca en la mitad ya que es una caja de dialogo

This is an open dialog window

4. Details o Detalles

Propósito: Define detalles adicionales que el usuario puede ver o esconder.

Sintaxis:
<details>
<summary>Copyright 1999-2018.</summary>
<p> - by Refsnes Data. All Rights Reserved.</p>
</details>

Atributos: Este elemento incluye los atributos globales por lo que no posee atributos específicos.

Copyright 1999-2018.

- by Refsnes Data. All Rights Reserved.

All content and graphics on this web site are the property of the company Refsnes Data.

11. Medidor

Propósito:Define una medición escalar dentro de un rango conocido (un medidor).

Sintaxis
<meter value="2" min="0" max="10"> 2 out of 10 </meter>

Atributos:
-min especifica el valor mínimo. -max especifica el valor maximo. -value especifica el valor actual del medidor.

Despliega un medidor:

2 out of 10
60%

Sintaxis:
<p>
This is a veryveryveryveryveryveryveryveryveryveryveryveryveryvery<wbr>
longwordthatwillbreakatspecificplaceswhenthebrowserwindowisresized
</p>

Atributos: Este elemento incluye los atributos globales por lo que no posee atributos específicos.

This is a veryveryveryveryveryveryveryveryveryveryveryveryveryveryveryveryveryverylongwordthatwillbreakatspecificplaceswhenthebrowserwindowisresized.

Las anotaciones ruby son trozos cortos de texto que corren a la par del texto base y proveen algún tipo de ayuda para procesarlo. Son mayormente utilizadas en la tipografía asiática, pero pueden ser útiles para cualquier tipo de anotaciones.

Sintaxis:
<ruby>

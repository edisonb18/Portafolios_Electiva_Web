Propiedadede de CSS

1. background-image:url("Imagenes/fondo1.jpg");

Próposito: Funciona para añadir una imagen como fondo de pantalla del sitio.

Rango:Se pueden usar la imagen que el desarrollador prefiera.

El ejemplo es de la página actual que posee un fondo gris con cambios de tono

2. background-size;

Descripción:Esto funciona para declarar un tamaño de la imagen de fondo, si la imagen es más grande o pequueña, esto la ajustará.

Rango: El rango va desde 0% hasta 100%

3. font-family: sans-serif;

Descripción:Esta propiedad determina una fuente de letra. En este caso san serif

Rango: Existen muchos tipos de fuente entre los que se pueden nombrar: Arial, Helvetica, Times New Roman, etc.

El ejemplo se encuentra en toda la pagina, donde la letra se cambia por defecto a sans serif.

4. Color

Descripción: Esta propiedad hace que la letra sea del color que el desarrollador elija.

Rango: Esta propiedad maneja una gran gama de colores, casi cualquiera que se quiera.

Texto en rojo

5. background-color: coral;

Descripción: Esta propiedad puede cambiar el color del fondo del sitio, no es una imágen.

Rango:Esta propiedad maneja una gran gama de colores, casi cualquiera que se quiera.

El fondo es verde obscuro

6. text-align;

Descripción: Esta propiedad alinea el texto al lugar que se desee.

Rango: left|right|center|justify|initial|inherit

text-align: center:

text-align: left:

7. border:border-width border-style border-color

Descripción: Esta propiedad define el color, el ancho y el tipo borde de las secciones, titulos, etc.

Rango: -El color en esta propiedad maneja una gran gama de colores, casi cualquiera que se quiera, -El ancho puede ser cualquier entero positivo y el tipo los valores pueden ser none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset|initial|inherit.

A heading with a dotted blue border

8. border-radius

Descripción:Esta propiedad sirve para rendondear las esquinas de una sección, etc.

Rango: El rango es cualquier numero entero positivo pero despues de 50 ya no se nota mucho la diferencia.

El ejemplo es la sección de esta página, donde los bordes son más redondos.

9. width

Descripción: Funciona para definir el ancho de una sección, imágen, etc.

Rango: El rango es cualquier numero entero positivo.

width: 150px

El ancho es muy pequeño.

10. height

Descripción: Funciona para definir el alto de una sección, imágen, etc.

Rango: Cualquier entero positivo.

height: 50px

El alto es bastante pequeño y no pasa de ahí

11. Margin

Descripción: Utilizada para crear espacio alrededor de los elementos

Rango: auto, length, %, inherit

Parrafo sin margenes.

Este parrafo tiene un margen de 35 en los 4 lados.

12. padding

Descripción: Se utiliza para generar espacio alrededor del contenido de un elemento

Rango: length, %, inherit

Este parrafo tiene un padding de 35 pixeles.

13. font-size

Descripción:Establece el tamaño del texto

Rango: px, em, %

15. float

Descripción:Especifica cómo debe flotar un elemento, en este ejemplo la imágen se corre a la derecha

Rango:none, left, right, both, inherit

16. opacity

Descripción:Especifica la opacidad / transparencia de un elemento.

Rango: numero entre 0 y 1

17. list-style-type

Descripción:Especifica el tipo de marcador del elemento de lista.

Rango:circle, square, upper-roman, lower-alpha, gif.

18. list-style-position

Descripción:Especifica la posición de los marcadores de elementos de lista

19. object-fit

Descripción:Especifica cómo se debe cambiar el tamaño de una img o video para que se ajuste a su contenedor

Rango:fill, contain, cover, scale-down, none

20. column-count

Descripción:Especifica el número de columnas en las que se debe dividir un elemento.

Rango: Enteros positivos.
